# Review

## Purpose

The **Review** folder contains processed documents that require human verification before becoming part of the permanent KnowledgeVault.

This stage provides an opportunity to validate AI-generated content, correct errors, improve metadata, and ensure that information meets the project's quality standards.

The Review folder acts as Ariadne's quality assurance checkpoint.

## Typical contents

Examples include:

- Processed documents awaiting approval
- AI-generated summaries requiring verification
- Notes with uncertain metadata
- Documents requiring manual classification
- Files with unresolved warnings
- Content flagged for further investigation

## Workflow

Documents typically arrive here after successful processing.

During review they may be:

- Approved and incorporated into the Wiki.
- Updated with improved metadata.
- Corrected where processing errors are identified.
- Returned for reprocessing if necessary.
- Moved to the Failed folder if significant issues are discovered.
- Archived once their processing lifecycle is complete.

The goal is to ensure that only verified knowledge becomes part of the permanent KnowledgeVault.

## What does *not* belong here?

This folder should not contain permanent Wiki articles or raw source material.

It exists solely as a temporary stage within the knowledge processing pipeline.

## Repository

The contents of this folder are intentionally excluded from the public repository.

Only this README file is included so that the folder structure is preserved when the project is cloned.

Each user's KnowledgeVault will populate this folder automatically as documents move through the verification process.

## Design Principle

Ariadne values trustworthy knowledge over rapid automation.

Human review provides an essential safeguard, ensuring that knowledge is accurate, complete, and reliable before it becomes part of the permanent KnowledgeVault.

---

**In short:**

The **Review** folder is Ariadne's quality assurance desk, where processed information earns the right to become trusted knowledge.
