# Ariadne Visual Roadmaps

These diagrams are the visual engineering documentation for the Ariadne project.

Unlike traditional software documentation, these boards are intended to communicate the project's direction at a glance. They act as engineering references, planning tools, and progress checkpoints throughout development.

## Documents

### 1. Ariadne Roadmap Overview

The master vision for the project.

This roadmap defines the five major stages of Ariadne's evolution, from a simple document library to a trusted AI companion.

Stages include:

- Build the Foundation
- Build the Memory
- Build the Workspace
- Build the Network
- Build the Companion

This document changes only when the long-term vision changes.

---

### 2. Ariadne Build Board

The active engineering board.

This document tracks the current implementation of Stage 1 and breaks the work into small engineering milestones.

It answers one question:

> What are we building next?

This board is updated regularly as milestones are completed.

---

### 3. Retired Engineer's Chaos Board

The reality check.

This is a humorous reminder that Ariadne is being built by one retired engineer, in between coffee, markets, road trips, YouTube, gaming, lunches with friends, and simply enjoying life.

The project is intentionally designed to grow through small, consistent improvements rather than full-time development.

---

## Philosophy

Plan first.

Document second.

Build third.

These diagrams keep the project focused, communicate progress visually, and provide a shared reference before writing code.

One retired engineer.

One AI system.

One step at a time.
